# Card Game Telegram Bot
**Author:** Natalia Kondratenko

## About the Project
This is a Telegram bot built with **Spring Boot** that lets you play a simplified **21-card game** against the dealer.  
The goal is to beat the dealer by getting closer to 21 without going over.

## Technologies Used
- **Spring Boot**
- **Telegram API**
- **Maven**
